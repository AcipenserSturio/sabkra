# sabkra
A .civ5map previewer that doesn't require you to load the whole game!

## Requirements

* Python >=3.11

* pdm (`pip install pdm`).

## Installation and usage

`pdm install`

`pdm run start`

You may also try `pdm run start -s` for a no-gui mode.
